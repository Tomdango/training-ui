What is React?

React.js is a JavaScript library developed by engineers at Facebook. It was developed as a scalable UI library for use in-house for Facebook.com.

There are a few reasons why you'd want to use React.

Why use React?

React is fast

React can handle complex page updates whilst feeling quick and responsive. This makes it a good choice for data-driven user interfaces and features.

React is modular

React allows for a modular, loosely-coupled structure for your application. This means having many smaller, reusable files.
